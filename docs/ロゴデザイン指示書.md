# Torch ロゴデザイン指示書

## システム概要

**Torch** は、SES企業向けの社内案件・人材情報管理システムです。

### システムの目的
- パートナー企業から毎日約2万件受信するメールから、案件情報と人材情報を自動抽出
- AIを活用して機密情報（商流、単価など）を隠蔽しつつ、必要な情報のみを社内に公開
- 市場分析やエンジニアへの情報共有を効率化

### 主要機能
- 案件情報の一覧・検索・詳細閲覧
- 人材情報の管理
- スキルタグによる検索・フィルタリング
- 市場分析機能

### ターゲットユーザー
- 社内スタッフ（@luxy-inc.com ドメインのみアクセス可能）
- エンジニア、営業、マネージャーなど

## ロゴデザインのコンセプト

### ネーミングの由来
**Torch（たいまつ）** = 暗闇を照らし、道を導く光

### コンセプトキーワード
1. **情報を照らす** - 大量のメールから必要な情報を抽出し、見やすく提示
2. **導く** - 適切な案件・人材とのマッチングを支援
3. **信頼性** - ビジネスツールとしての堅実さと信頼感
4. **モダン** - 最新技術を活用した洗練されたシステム

### デザインの方向性
- **プロフェッショナル**: ビジネスツールとして信頼できる印象
- **モダン**: 最新のWeb技術を感じさせる洗練されたデザイン
- **シンプル**: 機能性を重視し、過度に装飾的でない
- **識別性**: 一目で「Torch」と認識できる

## ロゴデザイン要件

### 基本仕様
- **形式**: SVG（ベクター形式）推奨
- **サイズ**: 最小32px高さで視認性を確保できること
- **カラーバリエーション**: 
  - フルカラー版（メイン）
  - モノクロ版（必要に応じて）
  - ダークモード対応版（オプション）

### デザイン要素の提案

#### アイコン要素
以下のいずれか、または組み合わせを検討：

1. **たいまつ（Torch）**
   - シンプルで洗練されたたいまつのアイコン
   - 炎の部分は情報や光を表現
   - 過度にリアルではなく、アイコンとして機能する抽象度

2. **光・ビーム**
   - たいまつから発せられる光をモチーフに
   - 情報の流れや検索を表現
   - 幾何学的でモダンなデザイン

3. **検索・照らす**
   - たいまつと検索の概念を組み合わせ
   - 情報を「照らし出す」という意味を視覚化

#### タイポグラフィ
- **フォント**: モダンで読みやすいサンセリフ体
- **スタイル**: ボールド（太字）で存在感を
- **配置**: アイコンの右側、または下側

### カラーパレット提案

#### メインカラー
- **オレンジ/アンバー系**: 炎・光を連想させる温かみのある色
  - 例: `#F59E0B` (Amber-500), `#F97316` (Orange-500)
- **ブルー系**: 信頼性・プロフェッショナル感を表現
  - 例: `#3B82F6` (Blue-500), `#2563EB` (Blue-600)

#### アクセントカラー
- **赤/オレンジ系**: 炎の部分に使用
  - 例: `#EF4444` (Red-500), `#F97316` (Orange-500)

#### グレースケール
- **ダークグレー**: テキストやアイコンの一部
  - 例: `#111827` (Gray-900), `#374151` (Gray-700)

### レイアウト
- **横長レイアウト**: アイコン + テキスト（推奨）
- **縦長レイアウト**: アイコンのみ、またはアイコン + テキスト（オプション）

## 使用シーン

### 主な使用場所
1. **Webアプリケーションのヘッダー**
   - 高さ32-40px程度のサイズ
   - 白背景またはグレー背景

2. **ファビコン**
   - 16x16px, 32x32px サイズでも視認可能
   - シンプルなアイコンのみでも識別可能

3. **ローディング画面・スプラッシュ画面**
   - より大きなサイズでの表示

### 制約事項
- 小さなサイズでも視認性を確保
- 白背景・グレー背景の両方で使用可能
- モバイルデバイスでも見やすい

## デザインスタイル参考
